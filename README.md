# BRAIN

## .put(status())

returns location, color and number of card as dictionary
### location:
	0: board
	1: discard pile
### color:
	the color as string
### number:
	the number of the card, 1 for multiplicator

## .draw(status())
### location:
	0: board
	1: discard pile
### color:
	the color of the discard pile from which to take
